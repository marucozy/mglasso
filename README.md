
# Meta Graphical Lasso

Estimate common latent factors across multiple datasets by assuming different sparse dependencies among the latent factors between datasets.


## Usage/Examples

### Run example code
```bash
./run.sh
```

### Create new data
```bash
./create_data.sh
```
